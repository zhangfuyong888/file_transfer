# LAN File Transfer - 局域网文件传输工具

在同一局域网内，通过浏览器进行文件传输的工具。支持文本、图片、文件等任意类型。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python3 transfer.py
```

默认端口 5000，可通过 `-p` 指定端口：

```bash
python3 transfer.py -p 8080
```

### 3. 打开网页

启动后终端会显示地址，例如 `http://192.168.1.100:5000`。

- **发送方**和**接收方**都在浏览器中打开该地址
- **发送方**切换到「发送 Send」标签页，拖拽文件或粘贴文本
- **接收方**切换到「接收 Receive」标签页，查看和下载文件

## 功能

| 功能 | 说明 |
|------|------|
| 📎 文件传输 | 拖拽或选择文件上传，支持超大文件分块上传（最大 2GB） |
| 💬 文本消息 | 发送文本、代码、笔记等 |
| 📋 剪贴板粘贴 | 在页面 Ctrl+V 直接发送剪贴板内容（文字/图片） |
| 🔍 LAN 扫描 | 点击扫描自动发现局域网内其他传输服务 |
| 📥 批量下载 | 接收方可一键下载所有文件 |
| 🔄 实时通知 | 新文件到达时自动推送通知 |
| 🌓 深色模式 | 自动适配系统深色/浅色主题 |

## 命令行参数

```
python3 transfer.py [选项]

选项:
  -p, --port PORT    服务端口 (默认: 5000)
  --no-cleanup       禁用过期文件自动清理
  --no-discovery     禁用局域网发现功能
```

## 注意事项

- 发送方和接收方需要连接到**同一局域网**
- 如果接收方无法访问，请检查发送方防火墙是否开放了对应端口
- 上传的文件默认在服务端保留 60 分钟后自动清理
- 服务停止后所有文件会被保留在 `uploads/` 目录中
