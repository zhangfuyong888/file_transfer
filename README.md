# LAN File Transfer - 局域网文件传输工具

在同一局域网内，通过浏览器进行文件传输。支持文本、图片、任意文件。

## 🚀 快速开始

### 方式一：双击启动（推荐）

| 系统 | 操作 |
|------|------|
| **Linux** | 双击 `start.sh`，或在终端运行 `bash start.sh` |
| **Windows** | 双击 `start.bat` |

程序会自动安装依赖、启动服务、并打开浏览器。

### 方式二：命令行启动

```bash
pip install -r requirements.txt
python3 transfer.py
```

### 方式三：Linux 桌面快捷方式

```bash
bash installer/install_desktop.sh
```

安装后桌面会出现快捷方式图标，双击即可启动。应用菜单中也会注册（搜索"局域网"即可找到）。

## 📖 使用说明

1. 发送方启动程序，浏览器自动打开页面
2. 发送方切换到**「发送 Send」**标签页
3. 接收方在浏览器中输入发送方显示的地址（如 `http://192.168.1.100:5000`）
4. 接收方切换到**「接收 Receive」**标签页
5. 发送方拖拽文件 / 粘贴文本 → 接收方实时看到并下载

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 📎 文件传输 | 拖拽或选择文件上传，大文件自动分块（最大 2GB） |
| 💬 文本消息 | 发送文本、代码、笔记等 |
| 📋 剪贴板粘贴 | 页面内 Ctrl+V 直接发送剪贴板内容 |
| 🔍 LAN 扫描 | 自动发现局域网内其他传输服务 |
| 📥 批量下载 | 一键下载所有文件 |
| 🔄 实时通知 | 新文件到达时自动推送 |
| 📅 日期归档 | 文件按日期自动分目录存储 |
| 🌓 深色模式 | 自动适配系统主题 |

## ⚙️ 命令行参数

```
python3 transfer.py [选项]

  -p, --port PORT    服务端口 (默认: 5000)
  --no-browser       不自动打开浏览器
  --no-cleanup       禁用过期文件自动清理 (默认 60 分钟)
  --no-discovery     禁用局域网发现功能
```

## 📁 文件存储

```
uploads/
├── 2026-07-11/          ← 按日期分目录
│   ├── 161818.txt       ← 文本消息：时间戳命名
│   ├── photo.jpg        ← 上传文件：保留原名
│   └── document.pdf
├── 2026-07-10/
│   └── ...
```

## ⚠️ 注意事项

- 发送方和接收方必须在**同一局域网**
- 浏览器地址必须带 `http://` 前缀（不要用 `https://`）
- 如接收方无法访问，请检查发送方防火墙是否开放了对应端口
- 文件默认 60 分钟后自动清理
